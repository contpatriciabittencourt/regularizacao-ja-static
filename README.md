# Regularização Já - Patrícia Bittencourt

Página web profissional para escritório de contabilidade especializado em regularização empresarial.

## Tecnologias

- React 19
- Vite
- Tailwind CSS 4
- TypeScript

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Conecte seu repositório ao Netlify para deploy automático.
